Function Explanation

ascii2sjis	Converts the ASCII code to the Shift-JIS code

	Syntax	int ascii2sjis( ascii_code )
		unsigned char ascii_code;

	Argument
		ascii_code	ASCII code

	Explanation
		This function converts the ASCII code to the Shift-JIS code.

	Return value
		On success, the function returns Shift-JIS code.
		On Failure, it returns 0.
